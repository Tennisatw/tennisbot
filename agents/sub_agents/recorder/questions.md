# 原生家庭
- 父亲“管得严”具体管什么、用什么方式管；小时候你最恨哪条规矩。
- 你和母亲更像盟友还是更像调停者；你小时候最常听到的评价词是什么。

# 关系与边界
- 你心里“自己人”的门槛是什么；朋友做什么会被你直接拉黑。
- 你能原谅的背叛是什么样；不能原谅的又是什么样。

# 友谊样本
- 你最稳定的三段友谊怎么开始、靠什么维持。

# 成就感/羞耻感/比较
- 第一次明显感到“我不如别人”的场景：什么时候、谁在场、你后来怎么消化。
- 你真正骄傲的一次胜利（不一定学业）。

# 过往经历（想讲清楚的两条主线）
- 你的人生里，哪三个时间点算“转折点”？每个转折点之前你在怕什么、之后你在追什么？
- 你觉得自己“日子变好”的关键节点是哪一年哪几件事？当时你具体做了什么选择？

# 人际关系（想讲清楚的两条主线）
- 你的人际关系里，最重要的三种关系类型分别是什么（伴侣/挚友/同事/网友等）？你从每一种关系里最想得到的是什么？
- 你最想讲清楚的三段关系（不一定甜）：每段关系的开始、一次关键冲突、以及你最后学到的东西。
